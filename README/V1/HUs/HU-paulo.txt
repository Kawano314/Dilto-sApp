"Como cliente, quero visualizar a lista de produtos com preços atualizados, para decidir minhas compras antes de ir".
- O aplicativo deve exibir os produtos com nome, preço e categoria.
- Os preços exibidos devem ser os mesmos praticados no local.
- O aplicativo deve permitir atualizar os preços em tempo real.

"Como dono da conveniência, quero cadastrar e editar produtos e preços, para manter o catálogo atualizado".
- O aplicativo deve permitir incluir novos produtos com nome, preço e categoria.
- O aplicativo deve permitir editar preços e excluir produtos.

"Como cliente, quero ver promoções e descontos destacados, para aproveitar melhores ofertas".
- O aplicativo deve permitir marcar produtos como “em promoção”.
- O preço promocional deve aparecer junto ao preço original riscado.
- Os produtos em promoção devem ter destaque visual no catálogo.
