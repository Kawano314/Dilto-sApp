Horários de funcionamento

História 1 - Como cliente, quero visualizar se a conveniência está aberta ou fechada em tempo real, para não perder a viagem.

Critérios de aceitação:
- O sistema deve exibir o status “Aberta” ou “Fechada” e esse status deve ser atualizado automaticamente conforme o horário cadastrado.

História 2 - Como dono da conveniência, quero cadastrar os horários de funcionamento, para que os clientes tenham informações corretas.

Critérios de aceitação:
- O sistema deve permitir cadastrar horários de abertura e fechamento para cada dia da semana e permitir exceções (feriados ou eventos especiais).

História 3 - Como cliente, quero ver o horário de funcionamento semanal, para planejar quando ir até a conveniência.

Critérios de aceitação:
- O sistema deve exibir os horários de cada dia da semana e indicar o dia atual.
