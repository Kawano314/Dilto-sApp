Com base em ambos os relatos podemos ter possíveis funções adicionadas ao sistema:
Cadastro de Produtos (acompanhamento de estoque em tempo real)
- O usuário deve conseguir inserir nome, quantidade, data de validade, preço de venda e compra de um produto.
- O sistema deve permitir a edição e exclusão de produtos já cadastrados.
- O usuário deve poder definir uma quantidade minima para cada produto, emitindo um alerta quando o nível for atingido (ex: Brahma Lata → alerta quando < 10 u.)
- O dono deve poder gerar relatórios de vendas em períodos específicos (diário, semanal, mensal)
- O sistema deve apresentar relatórios e estoque com produtos mais vendidos e produtos parados.
